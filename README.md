## 001 数字にする

  - 具体例
    - case1
      - 入力: 1
      - 出力: 1
    - case2
      - 入力: 0.1
      - 出力: 0.1
    - case3
      - 入力: -1
      - 出力: -1
    - case4
      - 入力: a
      - 出力: 数字ではありません
