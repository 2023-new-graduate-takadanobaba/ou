## 001 数字にする
  > ユーザーが入力した文字列を数字に変換する。ただし、ユーザーが入力した文字列が数字でない場合は、`"数字ではありません"`と表示する。
  - case1
    - 入力: 1
    - 出力: 1
  - case2
    - 入力: 999
    - 出力: 999
  - case3
    - 入力: 0.1
    - 出力: 0.1
  - case4
    - 入力: a
    - 出力: please input numbers

## 002 3の倍数
  > ユーザーに入力させた数字が3の倍数かどうかを判定する。 3の倍数の場合は`"3の倍数です"`、違う場合は`"3の倍数ではありません"`と表示する。
  - case1
    - 入力: 4
    - 出力: 3の倍数ではありません
  - case2
    - 入力: 6
    - 出力: 3の倍数です
  - case3
    - 入力: 0.3
    - 出力: 3の倍数です
  - case4
    - 入力: abc
    - 出力: 数字ではないです。
